# MakerWorld Enhancements

This userscript provides various enhancements for the MakerWorld website, including quick search functionality and many more to come.

> [!IMPORTANT]
> This userscript moved to a new repository: [JMcrafter26/userscripts](https://github.com/JMcrafter26/userscripts)
>
> Please update your installation to the new URL:
>
> ```url
> https://raw.githubusercontent.com/JMcrafter26/userscripts/main/makerworld-enhancements/makerworld-enhancements.user.js
> ```
>
> The old repository will no longer receive updates.

## Features

- Quick Search: Search for projects on Printables or Thingiverse
- More features coming soon!

## Installation

### Prerequisites

- A userscript manager such as [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://www.greasespot.net/).
- On iOS devices, you can use [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) app to run userscripts.

### Via Greasyfork

[![Greasyfork](https://cdn.jsdelivr.net/gh/JMcrafter26/badges@main/src/assets/available/greasy-fork/cozy.svg)](https://greasyfork.org/en/scripts/560107-makerworld-enhancements)

You can find this userscript on [Greasyfork](https://greasyfork.org/en/scripts/560107-makerworld-enhancements). Simply click the "Install this script" button to add it to your userscript manager.

### Via GitHub

[![GitHub](https://cdn.jsdelivr.net/gh/JMcrafter26/badges@main/src/assets/available/github/cozy.svg)](https://raw.github.com/JMcrafter26/userscripts/main/makerworld-enhancements/makerworld-enhancements.user.js)

You can also install the userscript directly from this GitHub repository:

1. Copy the following URL:

   ```ur
   https://raw.github.com/JMcrafter26/userscripts/main/makerworld-enhancements/makerworld-enhancements.user.js
   ```

2. Open your userscript manager and choose the option to add a new script from a URL.
3. Paste the copied URL and confirm the installation.

#### iOS

1. Open the following URL in Safari on your iOS device:

   ```url
   https://raw.github.com/JMcrafter26/userscripts/main/makerworld-enhancements/makerworld-enhancements.user.js
   ```

   or [open this link directly](https://raw.github.com/JMcrafter26/userscripts/main/makerworld-enhancements/makerworld-enhancements.user.js).
2. In the extension menu, select "Userscripts" to install the script.
3. Confirm the installation in the Userscripts pop-up.

## Usage

Once installed, the userscript will automatically run on the Makerworld website. You can access its features through the provided interface elements.
